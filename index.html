Chain-of-Thought Recommendation Framework

Overview

Chain-of-Thought Recommendation (CoT-Rec) is an advanced recommendation framework that leverages Chain-of-Thought reasoning to enhance decision-making by incorporating step-by-step logical reasoning into the recommendation process. This approach significantly improves interpretability, adaptability, and performance across various recommendation scenarios.

The framework combines the power of Large Language Models (LLMs) with structured reasoning to provide more transparent and reliable recommendations.

Getting Started

Prerequisites

The framework requires Python 3.9+ and supports both CPU and CUDA-enabled GPU environments.

Installation

Clone the repository and set up the environment:

# Clone the repository
git clone https://github.com/Bowenislandsong/CoT-Rec.git
cd CoT-Rec

# Create conda environment from the provided configuration
conda env create -f llm_env.yml
conda activate llm_env

Quick Start

The framework includes several key components:

  • CoT Dataset Processing: Tools for preparing and processing Chain-of-Thought datasets
  • Ranking Models: Neural ranking heads and XGBoost-based rankers
  • Evaluation Metrics: Comprehensive metrics for assessing recommendation quality

Components

Chain-of-Thought Dataset Processing

The cot_dataset module provides tools for working with Chain-of-Thought reasoning data, including decoding strategies and task-specific processing for datasets like GSM8K.

Ranking Models

The framework includes multiple ranking approaches:

  • Neural Ranking Head: PyTorch-based neural network for learning ranking functions
  • XGBoost Ranker: Gradient-boosted decision tree approach for ranking
  • Custom Metrics: Specialized evaluation metrics for recommendation quality

Technology Stack

  • Deep Learning: PyTorch with CUDA support
  • Language Models: HuggingFace Transformers (Mistral, etc.)
  • Ranking: XGBoost, Neural Networks
  • Data Processing: Pandas, NumPy, PyArrow
  • Development: Jupyter Notebooks for experimentation
